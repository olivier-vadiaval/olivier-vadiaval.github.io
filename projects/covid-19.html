Tableau Viz: COVID-19 Data

Technologies used: Excel, Tableau
Dashboard/Tableau Viz: COVID-19 Dashboard by Olivier Vadiavaloo,
Datasets: Covid-19 Data Deaths and Vaccinations, Total Population by Country 2020

In this project, I create a dashboard (or Viz) in Tableau from the COVID-19 pandemic data (source: Kaggle Datasets) and the year 2020's population data (source: World Population Review). The COVID-19 dataset contains the number of cases, deaths and vaccinations by country and date - while the 2020 population dataset unsurprisingly contains the population size of each country in 2020.
